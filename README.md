# Installation
* Download [ZIP](https://github.com/diesphink/albert-themes/archive/master.zip)
* Copy .qss file to ```.local/share/albert/org.albert.frontend.boxmodel.widgets/themes/```
* Launch Albert & Change Theme to new theme
